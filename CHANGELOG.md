# Changelog

All notable changes to Orin.LAB are documented here.

## [1.0.0] - 2026-03-31

### Added
- Telegram AI Bot — Claude-powered crypto Q&A and signal delivery
- Signal Engine — AI-generated BUY/SELL/HOLD signals from on-chain data
- Market Analyst — deep market analysis using Claude Sonnet
- On-chain Agent — Solana wallet and transaction monitoring
- Auto Poster (TypeScript) — automated signal posting to Twitter/X
- TypeScript SDK — Solana price fetcher and signal client
- Signal CLI (Go) — lightweight command-line signal tool
- Core SDK (Rust) — high-performance signal computation and utilities
- GitHub Actions CI — Python, TypeScript, Go, and Rust checks
- CodeQL security scanning
- Dependabot for automated dependency updates
- Full test suite (Python unit tests + Rust unit tests)
