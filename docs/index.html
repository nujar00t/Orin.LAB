Setup

Up and running in minutes

Only 2 API keys required to start.

1
Clone & install
# Clone the repo
git clone https://github.com/nujar00t/Orin.LAB.git
cd Orin.LAB && make install
2
Configure environment
cp .env.example .env
# Add your keys:
TELEGRAM_BOT_TOKEN=your_token
ANTHROPIC_API_KEY=your_key
3
Run
python orin.py
# Or directly: make bot / make signal
